# Data Structures and Algorithms (University Projects)

This repository contains university practical works and laboratory assignments focused on fundamental computer science concepts, data structures, and algorithms.

---

## 🛠️ Tech Stack & Concepts
* **Language:** C / C++
* **Key Topics:** Memory management, pointer manipulation, complexity analysis ($O(n)$), and algorithm optimization.

---

## 📂 Repository Structure

Below is the list of implemented tasks and data structures:

| Folder / Project | Description | Key Concepts / Algorithms |
| :--- | :--- | :--- |
| `01-02_RAM` | Random Access Machine simulation | Low-level execution, basic arithmetic |
| `03_llist` | Singly/Doubly Linked Lists | Dynamic memory, node traversal, insertion/deletion |
| `04_lstack` | Stack implementation | LIFO (Last In, First Out) principle |
| `04_lqueue` | Queue implementation | FIFO (First In, First Out) principle |
| `05_graph` | Graph representations | Adjacency matrix/list, basic graph operations |
| `05_tree_arr` | Binary Trees | Array-based tree representation, binary search trees |
| `06_traverse` | Tree & Graph Traversal | BFS (Breadth-First Search), DFS (Depth-First Search) |
| `08_dnc_dp` | Divide & Conquer / Dynamic Programming | Matrix multiplication, optimal substructure, memoization |
| `09_sort1` | Basic Sorting Algorithms | Bubble Sort, Insertion Sort, Selection Sort |
| `10_sort2` | Advanced Sorting Algorithms | Radix Sort, Quick Sort / Merge Sort |

---

## 🚀 Key Takeaways & Skills Learned

Through these projects, I have developed a strong foundation in:
1. **Memory Management:** Working with pointers, dynamic allocation (`malloc`/`free` or `new`/`delete`), and preventing memory leaks.
2. **Algorithm Efficiency:** Understanding and analyzing Time and Space Complexity using **Big O notation**.
3. **Data Organization:** Choosing the right data structure (e.g., Hash Tables vs. Trees) to solve specific computational problems efficiently.
